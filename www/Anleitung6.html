    <h1>Willkommen zur Anleitung der Investitions-App</h1>

    <p>Diese Anleitung wird Ihnen helfen, die Funktionen und Möglichkeiten unserer App zu verstehen.</p>


    <h2>Übersicht</h2>

    <p>Auf der Übersichtsseite haben Sie die Möglichkeit, zwischen den Reitern "Portfolio" und "Kursübersicht" zu wählen. Unter "Portfolio" können Sie Ihr derzeitiges Portfolio sehen und ein neues anlegen. Auf der Kursübersicht können Sie die aktuellen Kurse aller verfügbaren Assets einsehen.</p>

    <img src="uebersicht.png" alt="Übersicht">


    <h2>Portfolio</h2>

    <p>Unter "Portfolio" sehen Sie eine Übersicht über Ihr derzeitiges Portfolio und die dazugehörigen Werte. Sie können hier auch ein neues Portfolio anlegen.</p>

    <img src="portfolio.png" alt="Portfolio">


    <h2>Investment</h2>

    <p>Unter "Investment" können Sie zwischen den Optionen "Rendite maximieren / Risiko minimieren", "Minimum Varianz Portfolio", "Tangentialportfolio", "Individuelles Investment" und "Kennzahlen" wählen. Hier können Sie Ihre Investments verwalten und optimieren.</p>

    <img src="investment.png" alt="Investment">


    <h2>Über uns</h2>

    <p>Hier finden Sie Informationen über unsere Geschichte, unser Team und unsere Mission.</p>


    <h2>Kontakt</h2>

    <p>Hier finden Sie unsere Kontaktdaten und unseren Standort.</p>


    <h2>Profil</h2>

    <p>In Ihrem Profil können Sie Ihre persönlichen Informationen einsehen und bearbeiten.</p>

    

    <h2>Rendite Maximieren / Risiko Minimieren</h2>

    <p>Unter "Rendite Maximieren / Risiko Minimieren" können Sie die Gewichtung Ihrer Assets so anpassen, dass Sie entweder Ihre Rendite maximieren oder Ihr Risiko minimieren können. Hier haben Sie auch die Möglichkeit, zu wählen, ob Sie Shorting zulassen möchten oder nicht.</p>

    <img src="rendite_risiko.png" alt="Rendite Maximieren / Risiko Minimieren">

    

    <h2>Minimum Varianz Portfolio</h2>

    <p>Unter "Minimum Varianz Portfolio" können Sie ein Portfolio erstellen, das das geringste Gesamtrisiko aufweist. Dies wird erreicht, indem die Gewichtung der Assets so angepasst wird, dass die Varianz des Portfolios minimiert wird.</p>

    <img src="minimum_varianz.png" alt="Minimum Varianz Portfolio">

    

 <h2>Tangentialportfolio</h2>

<p>Unter dem Reiter "Tangentialportfolio" wird Ihnen das Portfolio mit der höchsten erwarteten Rendite pro Einheit Risiko, d.h. mit der höchsten Sharpe-Ratio, angezeigt. Dieses Portfolio wird auch als Tangentialportfolio bezeichnet, da es den Punkt auf der sogenannten Effizienten Grenze darstellt, der tangent an die risikofreie Rate ist.</p>

<p>Wie Sie im Diagramm sehen können, ist die Rendite des Tangentialportfolios höher als die der anderen Portfolios, aber auch das Risiko ist höher. Die genaue Zusammensetzung des Tangentialportfolios in Bezug auf die verschiedenen Assets wird ebenfalls angezeigt.</p>

<img src="tangentialportfolio.png" alt="Tangentialportfolio">


    <h2>Individuelles Investment</h2>

    <p>Unter "Individuelles Investment" können Sie selbst entscheiden, in welche Assets Sie investieren möchten. Sie können die Höhe Ihrer Investments in jedes Asset selbst bestimmen.</p>

    <img src="individuelles_investment.png" alt="Individuelles Investment">


    <h2>Kennzahlen</h2>

    <p>Unter "Kennzahlen" finden Sie verschiedene statistische Kennzahlen, die Ihnen dabei helfen können, die Performance Ihres Portfolios zu beurteilen. Folgende Kennzahlen sind verfügbar:</p>

    <ul>

        <li>Gewinn (jährlich): Dies ist die durchschnittliche jährliche Rendite Ihres Portfolios.</li>

        <li>Risiko (jährlich): Dies ist das jährliche Risiko Ihres Portfolios, gemessen als Standardabweichung der Renditen.</li>

        <li>Sharpe Ratio: Dies ist das Verhältnis von Überschussrendite (Rendite über dem risikofreien Zinssatz) zu Risiko. Eine höhere Sharpe Ratio deutet auf eine bessere risikoadjustierte Performance hin.</li>

        <li>Sortino Ratio: Dies ist das Verhältnis von Überschussrendite zu Downside-Risiko (Risiko von negativen Renditen). Eine höhere Sortino Ratio deutet auf eine bessere risikoadjustierte Performance hin, wenn nur negative Renditen als Risiko betrachtet werden.</li>

        <li>Beta: Dies ist ein Maß für das systematische Risiko Ihres Portfolios. Ein Beta von 1 bedeutet, dass Ihr Portfolio im Durchschnitt genauso stark auf Marktveränderungen reagiert wie der Markt selbst.</li>

        <li>Treynor Ratio: Dies ist das Verhältnis von Überschussrendite zu Beta. Eine höhere Treynor Ratio deutet auf eine bessere risikoadjustierte Performance hin, wenn nur das systematische Risiko betrachtet wird.</li>

    </ul>
